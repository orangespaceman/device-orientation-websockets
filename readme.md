# Device Orientation - Websockets

## Installation

1. Install dependencies:

```
npm install
```

2. Run app:

```
node index.js
```

3. Open web pages:

 - Device: [http://localhost:7006/](http://localhost:7006/)
 - Host: [http://localhost:7006/host/](http://localhost:7006/host/)
